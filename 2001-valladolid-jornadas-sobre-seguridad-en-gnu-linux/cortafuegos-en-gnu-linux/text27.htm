Diferencias en netfilter

  • Implementación de IP masquerade y Network Address Translation

  • Se reduce el número de reglas necesarias para enrutar.

  • Paquetes de enrutamiento dejan de afectar a las cadenas de entrada y de salida.

  • Capacidad de extender las políticas de filtrado sin afectar al Kernel.

  • Dos tablas de reglas filter y nat

  • Cadenas para nat: PREROUTING, POSTROUTING

  • Compatibilidad con versiones anteriores mediante módulos.

  • Rangos de puertos
