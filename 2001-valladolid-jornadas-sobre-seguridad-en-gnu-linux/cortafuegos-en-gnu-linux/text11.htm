Cortafuegos software (1)

  • Filtrado de paquetes (Filtering Firewalls)

    • Tratamiento de paquetes

Notas:

The beginning of each packet that enters your network has a header, specifying information about that packet
Source and Destination IP addresses
Source and Destination Port Numbers
Protocol Type
In general, you set up “Firewall Rules”, which examine header information and either accept or deny the packet.
