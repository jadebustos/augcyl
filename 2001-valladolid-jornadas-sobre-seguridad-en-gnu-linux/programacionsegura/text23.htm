Entrada de datos: URL’s

  • Algunos caracteres están codificados como %HH. Por tanto deben ser decodificados previamente y después se deben realizar las validaciones.

  • La decodificación sólo se debe hacer una sola vez

    • %2500

  • El programa servidor debe realizar todas las validaciones y no fiarse que ya se hayan hecho en el cliente (con un javascript por ejemplo)

  • No usar métodos GET para realizar operaciones que no sean de consulta exclusivamente
