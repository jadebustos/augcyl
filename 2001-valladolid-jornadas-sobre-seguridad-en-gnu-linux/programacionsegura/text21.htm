Entrada de datos: Variables de Entorno

  • Solución general al problema de las variables de entorno

    • Leer las variables que se necesiten y validar tanto sus valores como su longitud
    • Borrar todas las entradas:
      • Poniendo a null la variable global
      • Usando la función clearenv (no documentada)
      • Usando un programa “wrapper”
    • Restablecer las variables necesarias a valores seguros
