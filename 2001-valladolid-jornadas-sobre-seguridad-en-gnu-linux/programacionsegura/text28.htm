Desbordamiento de Buffer

  • Consecuencias de sobre-escribir la dirección de retorno:

    • Si el valor es aleatorio, lo más probable es que obtengamos un error “Segmentation Fault” y que el programa “muera”
    • Si conseguimos colocar la dirección de código correcto, éste se ejecuta
    • Si además el programa es SUID o SGID y pertenede a root se podría ejecutar código arbitrario con permisos de administrador
