Desbordamiento de Buffer

  • Soluciones (1)

    • Evitar el uso de funciones de cadena sin validación de tamaño: strcpy(3), strcat(3), gets(3), familia de sprintf(3) y scanf(3)
    • Otras funciones problemáticas son: realpath(3), getopt(3), getpass(3), streadd(3), strecpy(3), strtrns(3), getwd(3)
    • Existen herramientas para depuración de estos y otros problemas como Purify, Code Center, ElectricFence, GCT. Requieren una colección de casos de prueba.
