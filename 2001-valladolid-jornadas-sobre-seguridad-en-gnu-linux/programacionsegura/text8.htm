Mecanismos de seguridad en UNIX

  • Si se ejecuta un fichero que tiene activados el SUID o el SGID, el proceso resultante se ejecutará con la identidad del propietario de ese fichero

  • Cuando se crea un fichero su propietario será el EUID del proceso que lo crea.

  • Este comportamiento, aplicado a programas no desarrollados con el suficiente cuidado, puede comprometer la seguridad de todo el sistema
