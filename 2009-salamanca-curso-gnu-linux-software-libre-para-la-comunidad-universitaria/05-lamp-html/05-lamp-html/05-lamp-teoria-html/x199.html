3.4. Instalación y configuración de una aplicación LAMP.

Debido a que la instalación y configuración específicas dependen del sistema operativo y la versión escojida, se van a enumerar los pasos necesarios para su configuración, que son los siguientes:

  • Creación de una base de datos en el servidor para la aplicación.

  • Colocación de las tablas principales necesarias para el funcionamiento de la aplicación LAMP.

  • Instalación de la aplicación web en el sitio web principal.

  • Configuración de la aplicación web.

  • Prueba de la aplicación web.
