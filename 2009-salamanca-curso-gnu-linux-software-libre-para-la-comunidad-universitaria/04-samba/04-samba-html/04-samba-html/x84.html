1.4. smbpasswd

Si se usa, se debe inicializar encrypt passwords = yes . Permite añadir usuarios al fichero de configuración /etc/samba/smbpasswd o cambiar la contraseña ya existente.

smbpasswd -a <usuario> añade un usuario
smbpasswd <usuario> cambia la contraseña de <usuario>
