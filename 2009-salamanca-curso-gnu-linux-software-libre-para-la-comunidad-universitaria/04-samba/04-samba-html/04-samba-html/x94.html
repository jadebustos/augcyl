1.6. Utilidad swat (Samba Web Administration Tool)

Existe la posibilidad de administrar Samba desde un navegador Web utilizando swat. Se consigue tecleando la siguiente URL:

http://<IPServidorWeb>:901

Cuando se arranque, esta página Web pedirá la contraseña del root.

Para activar swat, hay que acceder al YaST -> Centro de Control -> Servicios de red (inetd) y activar el servicio correspondiente.Asimismo, se debe modificar el siguiente fichero /etc/xinet.d/swat en su parámetro only_from de tal forma que apunte a la IP desde la cuál se va a administrar Samba.

Evidentemente, habrá que reiniciar xinetd tras hacer cambios en este fichero.
