Capítulo 9. Circuitos eléctricos

9.1. Logisim

Logisim es una herramienta educativa para el diseño y la simulación de circuitos lógicos digitales.

Aunque cuenta con una interfaz muy sencilla, Logisim puede ser usado (y es usado) para diseñar y simular completas unidades centrales de procesamiento con propósitos educativos.

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS entre otros.
