11.2. KStars

KStars es el planetario del proyecto KDE.

Ofrece una simulación precisa del cielo nocturno para cualquier localización terrestre a la fecha y hora deseadas. La representación incluye 130.000 estrellas, 13.000 objetos del cielo profundo, el Sol, la Luna y miles de cometas y asteroides.
