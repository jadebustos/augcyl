2.4. OpenOffice.org Math

OpenOffice.org Math es el editor de ecuaciones de las suite ofimática OpenOffice.org.

Aparece en esta recopilación por su capacidad para exportar las ecuaciones en formato MathML, que es el estándar para incluir expresiones matemáticas en páginas web.

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS entre otros.
