2.2. GNU TeXmacs

GNU TeXmacs es un editor WYSIWYG (lo que ves es lo que consigues) especilizado en textos científicos.

GNU TeXmacs es capaz de ejecutar de manera integrada a aplicaciones como gnuplot, Maxima, GNU Octave...

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS entre otros.
