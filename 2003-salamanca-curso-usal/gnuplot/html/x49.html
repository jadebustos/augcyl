1.2. Iniciar y finalizar gnuplot

La forma más común de utilizar gnuplot es de forma interactiva en un entorno gráfico. Para el caso de Linux, desde las X, en un terminal teclear "gnuplot". Aparece un mensaje de saludo y el prompt de gnuplot.

Para salir basta con teclear "quit" o Ctrl-D.

El sistema de ayuda en línea de gnuplot se invoca desde el prompt de la aplicación con el comando "help" seguido opcionalmente por el comando u opción de la que se quiere información.

gnuplot posee multitud de opciones cuyos valores se pueden consultar con el comando "show" y se pueden cambiar con el comando "set". Para más información sobre todos las opciones disponibles, teclear "help set".
