2.8. Editor de gráficos vectoriales

2.8.1. Inkscape

Inkscape es un editor de gráficos vectoriales que utiliza como formato el estándar del World Wide Web Consortium llamado SVG.

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS.
