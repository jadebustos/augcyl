2.11. Programas de intercambio P2P

2.11.1. aMule

aMule es un programa de intercambio P2P similar al popular eMule que funciona con la redes eDonkey y Kademlia.

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS entre otros.

2.11.2. Azureus

Azureus es un cliente de la red BitTorrent.

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS.
