Capítulo 10. Tabla periódica de los elementos

10.1. Kalzium

Kalzium es una aplicación que muestra la tabla periódica de los elementos.

Además de los datos comunes que aparecen en las tablas impresas, dispone de otros tales como el punto de fusión, la afinidad electrónica, el radio covalente, el origen del nombre...
