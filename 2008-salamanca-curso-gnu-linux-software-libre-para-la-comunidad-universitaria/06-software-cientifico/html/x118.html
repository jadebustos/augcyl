2.3. LyX

LyX es un procesador de texto gráfico basado en el modelo WYSIWYM (lo que ves es lo que quieres decir). Esto significa que el autor sólo se debe preocupar de la estructura y contenido del texto, ya que el programa se encarga de la presentación.

La potencia de LyX radica en que combina la potencia de LaTeX con la facilidad de uso de una interfaz gráfica.

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS.
