3.3. Dibujos geométricos

3.3.1. GeoGebra

GeoGebra es un sistema de geometría dinámica.

Puede hacer construcciones con puntos, vectores, segmentos, secciones cónicas, funciones...

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS entre otros.

Figura 3-5. GeoGebra

3.3.2. GEONExT

GEONExT es un programa para hacer dibujos geométricos que pueden ser animados.

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS.
