5.2. FreeMat

FreeMat es un entorno para científicos e ingenieros que se dedican al modelado y procesamiento de datos.

Su sintaxis es altamente compatible con la del sistema privativo MATLAB.

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS.
