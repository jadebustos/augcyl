6.3. Los datos: Alamacenado y Clasificados con Mergeant

Desde GNOME 2.0 las aplicaciones de gASQL y GNOME-DB se han mezclado en una sola aplicación, llamada Mergeant. Desde la cual podemos acceder a los datos de una base de datos para su modificación, el tipo de base de datos que maneja es amplia, su funcionamiento es mediante un sistema que podriamos llamar de plug-ins, así se podrán menjar bases de datos MySQL, Postgres e incluso conectar con un servidor de SQL Server, aunque su implementación está aún muy verde.

La aplicación presenta una interface muy sencilla, y ademas, su integración con el escritorio es total. Desde el Centro de Control podemos gestionar las bases de datos y provedores que se disponene, en el menú de Preferencias del Escritorio, Avanzadas y allí en la opción de Configuración de la base de datos.
