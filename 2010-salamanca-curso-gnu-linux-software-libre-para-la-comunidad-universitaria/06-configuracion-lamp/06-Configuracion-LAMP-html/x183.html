3.3. Instalación y configuración de un Sistema de Gestión de Bases de Datos: Mysql.

Debido a que la instalación y configuración específicas dependen del sistema operativo y la versión escojida, se van a enumerar los pasos necesarios para su configuración, que son los siguientes:

  • Configuración básica del servidor Mysql: memoria principal disponible, tipos de tabla disponibles, memoria destinada a cache, etc...

  • Inserción de usuarios básicos.

  • Creación de una base de datos de ejemplo.

  • Creación de una tabla de ejemplo.

  • Modificación de un usuario para tener acceso a la base de datos creada.

  • Prueba de funcionamiento con un cliente de Mysql.
