Capítulo 6. Análisis estadístico

6.1. R

R es un entorno y lenguaje de programación para análisis estadístico y gráfico.

Se trata de una implementación libre del premiado lenguaje S.

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS entre otros.

6.1.1. R-Commander

R-Commander es una interfaz gráfica de usuario para R consistente en una ventana con menús, botones y campos de información.
