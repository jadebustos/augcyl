Capítulo 11. Simulación astronómica

11.1. Celestia

Celestia es un simulador espacial tridimensional, que al contrario que la mayoría de software astronómico no nos confina a la superficie de la Tierra.

Es un programa que ha sido utilizado por la NASA y la Agencia Espacial Europea. [1]

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS.
