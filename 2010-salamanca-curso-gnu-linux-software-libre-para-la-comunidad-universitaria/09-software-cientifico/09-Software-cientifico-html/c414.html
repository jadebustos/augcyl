Capítulo 12. Fractales e imágenes caóticas

12.1. GNU XaoS

GNU XaoS es un visor interactivo de fractales que permite al usuario hacer ampliaciones y reducciones de manera fluida y continua.

Puede generar una gran variedad de fractales de distintos tipos como Mandelbrot, Barnsley, Newton... También es capaz de guardar imágenes y vídeos de los fractales realizados.

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS entre otros.
