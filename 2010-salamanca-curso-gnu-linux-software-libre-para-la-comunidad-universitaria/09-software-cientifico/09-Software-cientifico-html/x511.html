A.2. Añadiendo repositorios

Hay veces que para poder instalar un programa paquetizado para Ubuntu hay que añadir repositorios adicionales.

A.2.1. Geogebra

Geogebra es un programa pensado para educación y esta enfocado a geometría.

root@host:~# echo "deb http://kondr.ic.cz/deb lucid main" >> /etc/apt/sources.list
root@host:~# wget http://kondr.ic.cz/kondr.key -O- | apt-key add -
--2010-10-28 22:53:18--  http://kondr.ic.cz/kondr.key
Resolviendo kondr.ic.cz... 88.66.103.249
Conectando a kondr.ic.cz|88.66.103.249|:80... conectado.
Peticion HTTP enviada, esperando respuesta... 200 OK
....
root@host:~# apt-get update

A partir de este momento ya podemos proceder a instalar Geogebra.
