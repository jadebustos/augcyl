Capítulo 7. Diseño asistido por computador o CAD

7.1. QCad

QCad es una aplicación para dibujo asistido por ordenador en dos dimensiones.

Con QCad se pueden crear dibujos técnicos como planos para edificios, interiores, piezas mecánicas o esquemas.

Es una aplicación multiplataforma disponible para GNU/Linux, Microsoft Windows y Mac OS entre otros.
