La lógica de negocio

  • La integración de sistemas, adaptación de aplicaciones y creación de programas se simplifica espectacularmente y hace que cada uno tenga la posibilidad de desarrollar sus propias ideas de negocio

  • Actualmente el proceso se realiza adaptando el "modus operandi" de la organización a las herramientas disponibles, el software libre cambia este concepto. Es posible adaptar o crear las herramientas conforme a como se quiere que funcione la organización
