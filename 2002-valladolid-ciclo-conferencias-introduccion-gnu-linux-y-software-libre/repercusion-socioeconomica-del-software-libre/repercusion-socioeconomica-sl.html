Que es el "software libre"

"Software Libre" se refiere a la libertad de los usuarios de correr, copiar, distribuir, estudiar, cambiar y mejorar el software. Mas precisamente, se refiere a las cuatro libertades de los usuarios de software:

(libertad 1)

  • La libertad de correr el programa, con cualquier propósito.

(libertad 2)

  • La libertad de estudiar como funciona el programa, y adaptarlo a sus necesidades.

    El acceso al código fuente es una precondición para esto.

(libertad 3)

  • La libertad de distribuir copias de manera que se puede ayudar al vecino.

(libertad 4)

  • La libertad de mejorar el programa, y liberar las mejoras al publico de tal manera que toda la comunidad se beneficia.

    El acceso al código fuente es una precondición para esto.
