4. Variables de entorno


Como dijimos antes, al servidor no sólo se le envía una petición de URL sino también una cierta información sobre el entorno en el que se pide y se ejecuta el script.

Como somos gente inquieta y queremos saber qué información es ésta, vamos a hacer un script que la devuelva

Este es el script que hace eso:

envvar.cgi
#!/usr/bin/perl

print("Content-type: text/html\n\n");

print("<html><head><title>Taller de Perl - CGI-Perl : Variables de entorno</title></head>\n");
print("<body bgcolor=white>\n");
print("<center><h1>Variables de entorno de %ENV{ }</h1></center>\n");
print("<hr WIDTH='100%'>\n");
print("<ul>\n");
foreach $variable (keys %ENV) 
{
     print("<li><font color=red>$variable</font> = $ENV{$variable}\n");
}
print("</ul></body></html>\n");

Como veis la información resultante es en algunos casos interesante...
