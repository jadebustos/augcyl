6. Un paso más: transferencia de ficheros


Por un socket, no sólo podemos mandar la hora, o mensajes de un protocolo, sino que podríamos por ejemplo transferir un fichero almacenado en disco. Vamos a modificar el cliente y el servidor para que el cliente cuando se conecte al servidor pueda obtener un fichero situado en el mismo directorio en el que se ejecuta el servidor.

El servidor (servidor2.pl) se encuentra en el directorio usual y el cliente (cliente2.pl) en un subdirectorio llamado file_transfer.

En el cliente, se abre una consola en la que se pueden introducir dos comandos:
 
DIR Lista los ficheros en el directorio del servidor
GET    <lista ficheros> Obtiene cada uno de los ficheros que se le indiquen
