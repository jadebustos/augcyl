3. El hola mundo en CGI


Como es ya una tradición empezaremos mostrando un script de cgi que manda al cliente un hola mundo
 

hola.cgi
#!/usr/bin/perl

print "Content-type: text/html \n\n";
print ("<!-- Esta pagina fue generada de manera dinámica -->\n");
print ("<html>\n");
print ("<head>\n");
print ("<title>\"Taller de Perl - CGI-Perl: Hola mundo\"</title>\n");
print ("</head>\n");
print ("<body bgcolor=white>\n");
print ("<center>");
print ("<h1><font color=red>Hola mundo</font></h1><br>\n");
print ("<h3><font color=blue>Este es mi primer CGI en Perl</font></h3>\n");
print ("</center>");
print ("</body>\n");
print ("</html>\n");

Como vemos, el script imprime por salida estándar una página html, lo que ocurre es que esta página es recogida por el servidor y mandada al cliente.

Es importante que la línea en verde  esté en todo script. Se permiten otras líneas si lo que se manda es otra cosa en vez de una página html (por ejemplo una imagen, un archivo de sonido, ...)
