5. Estructuras de control (página man: perlsyn)


Perl dispone de las estructuras de control habituales en otros lenguajes de programación. Se dividen en dos grupos: estructuras condicionales y estructuras iterativas.
 

Condicionales
 
if ( expr. booleana )
{
    ...
}

 
if ( expr. booleana )
{
    ...
}
else
{
    ...
}

 
if ( expr. booleana )
{
    ...
}
elsif ( expr. booleana )
{
    ...
}
...
else
{
    ...
}

 
unless ( expr. booleana )
{
    ...
}

expr. booleana son expresiones lógicas que pueden ser compuestas, usando operadores lógicos.

Algunos de los operadores lógicos para las expr. booleanas son:   (página man: perlop)
 
AND &&
OR  |   |
NOT !
igual  ==
eq		 (numeros)
(cadenas de caracteres)
distinto ! =
ne 		 (numeros)
(cadenas)

 

Iterativas
 
 
while  ( expr. booleana )         Se evalúa la condición al principio. Bucle   0+ veces
{
    ...
}

 
do
{
    ...
} while ( expr. booleana )     Se evalúa la condición al final. Bucle  1+ veces (Al menos se ejecuta siempre 1 vez)

 
for($i = ini; $i < fin ; $i++)            Bucle con número de iteraciones fijo igual a (fin-ini)
{
    ...
}

 
foreach $value(@array)            Bucle con número de iteraciones fijo e igual al número de elementos del array
{ 
    ...                                                 $value toma el valor de un elemento del array diferente en cada iteracion
}
