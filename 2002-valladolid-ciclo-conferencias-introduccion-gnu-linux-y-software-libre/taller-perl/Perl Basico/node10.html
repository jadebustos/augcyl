11. Modulos CPAN

En Internet podemos encontrar una gran cantidad de módulos hechos por programadores de todo el mundo y que resuelven todo tipo de problemas.

Los hay muy usados como

IO
NET
FTP
XML
CGI

Usualmente suelen estar empaquetados en un fichero tar.gz o .tgz y su instalación es sencilla. Suele reducirse (si son estandar) a:

1. descomprimir el tar.gz,

2. Leer el README por si acaso (siempre es buena política)

3. Generalmente hay un fichero llamado Makefile.pl. Hacer  perl Makefile.pl

4. Make

5. make install

A partir de aquí se pueden empezar a usarlos.
 

Se pueden bajar de www.cpan.org o de cualquiera de sus mirrors.
