Los módulos son ficheros independientes del script principal que pueden contener variables y subrutinas.

Suelen tener extension .pm

En el script principal se pueden usar con la directiva use

Para invocar a una subrutina definida en un módulo la notación es:

<nombre del módulo>::<subrutina>(parametros)
ejemplo15.pl
#!/usr/bin/perl -w

# Ejemplo que usa un modulo Math definido por nosotros (Math.pm)

use Math;
 

$M = Math::max(1,2,3,4,53,5,2,12,12,4);
print "$M\n";

$m = Math::min(1,24,5,56,6,2,20,0,134,4,4);
print "$m\n";

Math.pm
# Modulo de funciones matematicas no usuales
 

package Math;
 

sub max
{
   $maximo = $_[0];
   for($i=1; $i<$#_; $i++)
   {
      if($_[$i] > $maximo) { $maximo = $_[$i]; }
   }
   return $maximo
}
 

sub min
{
   $minimo = $_[0];
   for($i=1; $i<$#_; $i++)
   {
      if($_[$i] < $minimo) { $minimo = $_[$i]; }
   }
   return $minimo
}
 

1;
__END__

Como el array de los parámetros de entrada se llama @_ , la variable que contiene el número de elementos es $#_ , que es la que se ha empleado en los bucles for
